# My Backend Server

A simple backend-only web server using Node.js and Express.

## Features
- REST API endpoint: `GET /health` returns `{ status: "ok" }`

## Getting Started

### Prerequisites
- Node.js (v14 or higher recommended)
- npm

### Installation

```
npm install
```

### Running the Server

```
node server.js
```

The server will start on port 3000 by default.
